# Student Management System in C++

## Objective
To develop a console-based Student Management System in C++ that efficiently manages student records using file handling and menu-driven operations.

## Features
- Add Student Record
- Display Student Records
- Search Student by Roll Number
- Update Student Information
- Delete Student Record
- Persistent Storage using File Handling
- Menu Driven Interface

## Technologies Used
- C++
- Object-Oriented Programming
- File Handling

## How to Run

1. Compile the program

g++ student_management.cpp -o student

2. Run the executable

./student

## Project Structure

student_management.cpp
students.txt (generated automatically)
